# command windows:

```
$ ctrl + s (save)
$ win + r (run)
$ ctrl + ~ (terminal vs code)
```

# terminal:

```
$ exit (exit)
$ pwd (print working directory)
$ ls (list directory)
$ cd (change directory)
$ clear/ctrl + l (clear)
```

# command git:

```
$ git -v (--version)
$ git clone [ssh]
$ git status
$ git restore --staged [file]
$ git add --all
$ git commit -m "text"
$ git log
$ git push

```
